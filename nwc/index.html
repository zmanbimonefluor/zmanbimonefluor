Tool Panel RVT-2-NWC

Description

RVT to NWC export tools. Generates a container revit model with project parameters to run the automation. The generated Task Xml is used to import into Task Scheduler.

Buttons

  • Create NWC Container - Wizard to generate container revit model and task xml.
  • Create Params - Adds project parameters into the currently opened revit model.
  • Export NWC - Runs the Nwc export. The current revit model must have the required parameters.
