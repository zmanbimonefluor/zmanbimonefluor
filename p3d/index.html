Tool Panel P3D-2-RVT

Description

P3D to RVT conversion tools. Generates a container revit model with project parameters to run the automation. The generated Task Xml is used to import into Task Scheduler.

Buttons

  • Create P3D Container - Wizard to generate container revit model and task xml.
  • Create Params - Adds project parameters into the currently opened revit model.
  • Convert P3D - Runs the conversion. The current revit model must have the required parameters.
