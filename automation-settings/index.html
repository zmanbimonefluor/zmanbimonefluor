Tool Panel Automation Settings

Description

Automation Settings tools. Including this quick guide. Enable/Disable automation.

Buttons

  • Quick Guide - This quick guide.
  • Config - Adds project parameters into the currently opened revit model.
  • Automation OFF - Disable Automation. Allows opening of an AutomationModel container to modify/update parameters.
  • Automation ON - Enable Automation. [default]
